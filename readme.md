# Typescript-utils
1. npm install typescript -g
2. 项目目录下 命令：tsc
3. dist-compiler目录编译输出

### 1.pub-sub 发布/订阅模式封装
  1. 支持异步
  2. 支持一次性订阅
  3. 支持精准分发
### 2.loop 定时器封装
### 3.color-transform 颜色转换 
  1. rgb ---> hex 
  2. hex ---> rgb 
### 4.html-element-event 事件监听封装
### 5.storage 储存封装
### 6.converToTree 一级数据转换为树级数据
### 7.file 文件操作
  1. base64ConverBlob base64与blob的转换
